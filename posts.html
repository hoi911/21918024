Python 예제

오늘은 파이썬으로 윤년을 검사하는 프로그램을 작성합니다.

먼저 input() 함수로 사용자가 원하는 연도를 입력받습니다.
year라는 변수에 입력 값을 저장하고 해당 입력 값이 4와 100으로 나누어 떨어지는지 검사합니다.
                    if year % 4 == 0 and year % 100 == 0:

이것이 "참"일 경우 조건을 하나 더 세워줍니다.
                    if year % 400 == 0:
만약 이 조건까지 "참"이라면 해당 연도는 윤년이 됩니다.
거짓이라면 평년이 되겠죠.

첫 번째 조건문 밑에 elif문을 사용해서 다른 조건을 붙여 줍니다.
이번 조건은 4로만 나누어서 떨어지는지 입니다.
                    elif year % 4 == 0:
이 경우에는 무조건 윤년이 됩니다.

아래에 else문을 넣고 위 조건들이 모두 "거짓"일 경우 평년이라는 결과를 출력합니다.

                    
                        year = int(input("연도를 입력해주세요."))

                        if year % 4 == 0 and year % 100 == 0:
                            if year % 400:
                                print("윤년")
                            else:
                                print("평년")
                        elif year % 4 == 0:
                            print("윤년")
                        else:
                            print("평년")
